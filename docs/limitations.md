# Limitations

This project is a public-data screening model. It should not be used as a regulatory determination, engineering siting tool, legal finding, official risk assessment, or statement that any water system is unsafe.

Known limitations to track:

- ECHO SDWA data are refreshed periodically and are not real-time state records.
- Quarterly SDWIS snapshots require care to avoid double-counting system inventory records.
- Service area boundaries may be authoritative, modeled, missing, or geographically uncertain.
- County-level joins can mask within-county variation in vulnerability and drought exposure.
- SRF project records may not contain PWSIDs, so funding matching confidence must be explicit.
- Missing funding matches do not prove a system did not receive funding.
- Drought exposure is a resilience context variable, not a drinking water quality finding.
- Scoring weights are analytical assumptions for portfolio screening and require stakeholder review before operational use.
